## ğŸ”§ Configuration

### Firebase Setup

1. Go to Firebase Console
2. Create a new project
3. Enable Firestore Database
4. Go to Project Settings > General
5. Scroll down to "Your apps" and click the web icon
6. Copy the configuration and paste it in `src/firebase.js`

### Customize Content

- **Personal Information**: Update details in `src/components/Home.jsx`
- **Skills & Education**: Edit `src/components/About.jsx`
- **Projects**: Modify project array in `src/components/Projects.jsx`
- **Contact Info**: Update contact details in `src/components/Contact.jsx`
- **Colors**: Customize theme colors in `tailwind.config.js`
